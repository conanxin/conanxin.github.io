## 推荐体验流程

### 3 分钟快速体验

1. 打开页面，等待加载完成（约 2–5 秒）
2. 点击 **Best First Look** 预设按钮
3. 观察 30 秒呼吸运动
4. 点击 **Auto Cycle: ON**
5. 等待状态自动切换，阅读 Narrative 面板
6. 切换 **Side** 视角，关闭 Auto Rotate，静静观察

### 深入探索

- **Mechanism Explanation**: 理解内部凸轮机构
- **Structure Exploded**: 查看零件分解关系
- **Diagnostics**: 检查加载状态和技术细节
- **各状态切换**: 手动选择 Idle / Reading / Sleeping / Observer / Awake

## 常见问题

### 页面一直 Loading

- 检查浏览器控制台（F12 → Console）是否有红色错误
- 确保 WebGL 已启用（浏览器设置 → 高级 → 系统 → 使用硬件加速）
- 刷新页面重试
- STL 文件从本域加载，无需 CDN

### 模型不显示

- 确认处于 **Assembly**、**Parts**、**Exploded** 或 **Motion** 模式之一
- 在 Parts 区域检查零件复选框是否勾选
- 尝试 **Reset View** 或 **Fit to View**
- 检查 Diagnostics 面板中的 "STL Loaded" 计数

### 手机卡顿

- 页面会检测小屏并显示提示
- 开启 **Performance Mode**（关闭标签、自动旋转、运动路径）
- 推荐使用桌面浏览器获得完整体验

### WebGL 不可用

- 某些浏览器或设备可能不支持 WebGL
- 尝试 Chrome / Edge / Firefox 最新版
- 更新显卡驱动

## 已知限制

| 限制 | 说明 |
|------|------|
| **不是真实 AI** | 状态切换基于计时器，不是环境感知 |
| **状态机是解释性** | 预设序列，不是行为树 |
| **无传感器输入** | 不能真正"听到"或"看到" |
| **运动仍是简化** | 非物理仿真，是解释性动画 |
| **无记忆** | 不会记住昨天的状态 |
| **加载时间** | 9 个 STL 文件约 1–3MB，首次加载需要时间 |
| **移动设备** | 小屏体验受限，推荐桌面 |
