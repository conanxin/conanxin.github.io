项目说明

关于这部作品

《会演的人》改编自播客节目「情感收录社」第四期，原集标题为「烂杆闺蜜的混乱情史」。故事以第一人称回忆体讲述了一段被消耗的友谊：一个认真过头的人，遇上了一个善于利用热情的人。

改编说明

本作品在播客原声转写基础上，经历了以下处理流程：

① 播客音频转写（faster-whisper）
② 文学化改写：口语体转书面语，补充环境描写与心理描写
③ 去风险处理：所有人名、地名均做脱敏处理，删除可能引发现实联想的细节
④ 结构重组：从口述流水账重组为八章带章节节奏的叙事结构
⑤ V2 修订：统一人物称谓，修正时序矛盾，优化情绪节奏

人物处理原则

所有人物均为化名，与播客中提及的任何真实人物无直接对应关系。情节做了戏剧化处理，不代表播客嘉宾或作者对任何真实事件的评价。

技术栈

转写：faster-whisper + 可选 pyannote.audio 说话人分离
改编：本地 Hermes Agent 文学化流水线
发布：GitHub Pages + GitHub Actions 自动部署
