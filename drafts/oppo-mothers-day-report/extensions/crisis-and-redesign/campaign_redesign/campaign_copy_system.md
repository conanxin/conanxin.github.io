## C. 长文案（3版）

### 长文案版本 1：叙事风格

妈妈每天都在记录。
记录你的成长，记录家庭的日常，记录每一个值得庆祝的时刻。
但我们很少看见——她也有一件认真对待的小事，
一个坚持了很久的爱好，一个她提起时眼睛会发光的人或事。

这个母亲节，我们想邀请你一起：
用镜头，认真记录妈妈为自己的热爱发光的样子。

不是摆拍，不是仪式感，
只是她认真做自己喜欢的事时，最真实的样子。

**妈妈，也有自己的热爱。**

*OPPO — 记录每一个值得认真对待的时刻*

---

### 长文案版本 2：对话风格

你有没有问过妈妈：
"你有什么喜欢做的事吗？"

不是"你喜欢做什么给我们吃"，
不是"你喜欢怎么安排家庭"，
而是——她自己，有什么真正热爱的事？

也许她喜欢在阳台上种花，
也许她每天早上会写一会儿日记，
也许她有一首反复听的老歌，
也许她有一个一直想去的地方。

这个母亲节，我们希望看见她。
不是作为妈妈的她，而是作为她自己的她。

**妈妈，也有自己的热爱。**
用OPPO，记录她为自己发光的样子。

---

### 长文案版本 3：观察者视角

我们总是看见妈妈在照顾别人。
但我们很少看见——
她在照顾别人之前，是否也在认真照顾自己？

她有没有一件，
不需要服务于家庭、不需要被任何人解释，
只是因为自己喜欢，所以认真对待的事？

如果有，
请帮她记录下来。
因为这件事里的她，
是这个世界上独一无二的样子。

**妈妈，也有自己的热爱。**

## E. 海报文案（5组）

### 海报 1：音乐
**场景：** 妈妈在听音乐（耳机或音响）
**主文案：** 她也会为一段旋律，安静地认真很久。
**副文案：** 妈妈的热爱，不需要被解释，只需要被看见。

### 海报 2：运动
**场景：** 妈妈在清晨跑步或做瑜伽
**主文案：** 在照顾家人之前，她也在照顾自己的身体。
**副文案：** 她的热爱，让她成为更好的自己。

### 海报 3：写作
**场景：** 妈妈在夜晚写作或写日记
**主文案：** 夜深以后，她也有自己的宇宙。
**副文案：** 妈妈的故事，不只有家庭这一页。

### 海报 4：旅行
**场景：** 妈妈拿着相机或手机在旅途拍摄
**主文案：** 她拍过很多人，也值得被认真记录。
**副文案：** 用镜头，看见妈妈的光。

### 海报 5：朋友
**场景：** 妈妈和朋友们在一起开心聊天
**主文案：** 她的笑声，也属于她自己。
**副文案：** 妈妈不只是一个角色，是一个完整的人。

## I. 风险词禁止清单

以下词汇和表达**不得**出现在任何文案中：

| 禁止词/表达 | 原因 |
|------------|------|
| "两个老公" | 饭圈特定语境，母亲节语境下严重冒犯 |
| "穿婚纱" | 暗示性表达，在母亲节语境下不恰当 |
| "比爸爸更…" | 将父母置于竞争框架，制造家庭对立 |
| "追星妈妈" | 污名化母亲群体 |
| "恋爱脑" | 对母亲的贬低性标签 |
| "妈妈疯了" | 贬低性表达 |
| "少女心" | 将成熟女性幼儿化 |
| "比妈妈更爱偶像" | 将父亲/配偶置于竞争框架 |
| "不顾家" | 对母亲的负面评判 |
| "全职妈妈不工作" | 对母亲群体的偏见 |
