# 发布阻断清单 · Publication Blockers

**状态：** 当前为本地 noindex 草稿，不可直接公开发布  
**目标：** 明确阻断原因、清理路径、可保留内容

---

## 1. 当前不能直接公开发布的原因

### 🔴 阻断项（必须解决）

| 阻断原因 | 严重程度 | 说明 |
|---------|---------|------|
| 内部参考图来自网上来源 | 🔴 高 | 28 张下载的图片来自 UCCA 官网和第三方媒体，未经授权 |
| 部分素材仅供本地学习参考 | 🔴 高 | `internal_reference_only` 标记的素材不可公开使用 |
| 缺图作品仍是占位图 | 🟡 中 | 10 件作品使用 SVG 占位图，无真实作品图像 |
| 部分解读文本需要人工复核 | 🟡 中 | viewing_tip、related_works 等字段为推断内容 |
| 未确认 UCCA 展馆拍摄与发布规则 | 🟡 中 | 需要确认 UCCA 允许摄影及发布自摄照片的规则 |

### 🟡 风险提示（建议处理）

| 风险 | 说明 |
|------|------|
| 观众肖像 | 如果用户自摄照片中出现可识别的观众正脸，发布前需模糊处理 |
| 影片截图 | 不要大规模使用展览中播放影像的截图 |
| 商标/品牌 | 展厅中可能出现的赞助商标识（如 LV、Dior 等）需注意 |

- [ ] 确认 `.gitignore` 仍然排除这些文件

### 阶段五：技术准备
- [ ] 决定：保留 noindex 还是移除
- [ ] 决定：保留在 drafts/ 还是移出到正式路径

## 3. 可公开保留的内容

### ✅ 可自由使用

| 内容 | 原因 |
|------|------|
| 原创页面结构（HTML/CSS/JS） | 自制代码，无版权问题 |
| SVG 占位图（13张） | homemade，原创视觉 |
| 交互逻辑与数据管线 | 自制代码 |
| 官方资料的摘要性改写 | 合理使用，已标注来源 |
| 来源链接与 credit 信息 | 公开信息，合理引用 |
| 用户自摄或授权素材 | 用户拥有版权 |
| 观展笔记功能（localStorage） | 纯前端功能，无数据外传 |

### ⚠️ 需要条件保留

| 内容 | 条件 |
|------|------|
| UCCA 官方文字摘要 | 保留来源链接，不冒充官方 |
| 策展章节改写 | 注明"基于 UCCA 官方材料整理" |
| 艺术家履历 | 注明来源，不修改事实 |

## 5. 发布路径建议

### 方案 A：完全自摄版（推荐）
1. 用户现场拍摄全部 16 件作品 + 展厅环境
2. 删除所有网上参考图
3. 关闭 internalMedia 模式
4. 移除 noindex，移出 drafts/
5. 公开上线

### 方案 B：混合授权版
1. 获得 UCCA 或艺术家工作室的图片使用授权
2. 使用授权图 + 自摄图混合填充
3. 删除未授权网上图
4. 关闭 internalMedia 模式
5. 公开上线

### 方案 C：学习草稿版（当前状态）
1. 保持 noindex
2. 保持 drafts/ 路径
3. 不公开上线
4. 仅供个人学习回顾使用
5. 使用 internalMedia 模式预览网上参考图

**结论：** 当前页面作为本地学习草稿已完整可用。公开发布的路径清晰：删除网上图 → 填充自摄/授权图 → 关闭内部模式 → 核对信息 → 移除 noindex。最核心的问题是图片授权，而非代码或数据。
