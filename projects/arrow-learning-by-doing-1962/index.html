Kenneth J. Arrow
《干中学的经济含义》学习导读

经验、技术变迁与内生增长理论的一个起点

技术变迁不是凭空而降的外生力量，而是生产活动中内生地积累的产物——每当企业进行投资、雇佣劳动力、扩大生产规模，就会在"干中学"中积累经验，推动生产率提升。Arrow 的核心洞见是：这一知识积累过程具有显著的溢出效应，单个企业无法完全捕获其创造的社会收益，因此竞争性市场会导致总投资不足、社会最优路径偏离竞争均衡。这篇论文为后来 Romer（1986）、Lucas（1988）的内生增长模型奠定了基础，也是理解 AI 时代知识经济外部性的理论起点。

这篇论文在问什么？

20 世纪 50–60 年代，经济学主流增长理论将技术变迁视为外生给定——仿佛技术是从天上掉下来的，与经济系统的运行无关。Arrow 问了一个看似简单却深刻的问题：

如果技术变迁本身是由经济活动内生地产生的——比如企业通过投资和生产积累经验——那么这种内生化的技术变迁会如何改变我们对经济增长的理解？竞争性市场能否自发达到社会最优？还是会系统性地投资不足？

这是一个关于增长理论基础的问题，Arrow 用严谨的数学模型给出了回答。这篇论文发表于 1962 年，直到 1980 年代 Romer 和 Lucas 的内生增长模型出现，才真正成为主流宏观经济学的核心文献之一。

什么是"干中学"？

"干中学"（Learning by Doing）是 Arrow 对一种经济现象的概念化命名：

  • 经验来自生产活动： 企业每完成一次投资、生产一批产品、进行一次技术改造，都会积累关于如何更有效生产的知识。
  • 知识是副产品： 这些知识不是企业刻意研发的结果，而是生产的"副产品"——做事情的过程中自然学会了把事情做得更好。
  • 知识有溢出效应： 一个企业的经验不会完全留在该企业内部。工人流动、专利披露、行业观摩，都使知识传播到其他企业，产生正外部性。
  • 积累有路径依赖： 过去的投资水平决定了当前的经验存量，进而决定当前的生产率——发达经济与落后经济之间的差距会自我强化。

Arrow 的关键洞察在于：知识溢出意味着社会收益率高于私人收益率——企业对"干中学"的私人投入只考虑了自身收益，但社会还收获了外溢给其他企业的知识。因此，竞争性均衡的投资水平会低于社会最优水平。

阿罗的论证链条

Arrow 的论证分为三个层次，从观察到理论再到政策含义：

第一步：经验观察

Arrow 列举了大量经验证据——行业生产率数据、飞机制造业的学习曲线、钢铁行业的规模效应——来说明生产率的提升与累计生产经验高度相关，而不仅仅是资本积累的结果。

第二步：形式化模型

Arrow 构建了一个包含"干中学"的生产函数模型。核心设定：

  • 总生产函数中的技术水平 K 是累积投资的函数（体现型技术变迁，即 embodied technical change）；
  • 投资既增加资本，也增加知识——但知识的部分社会收益无法被进行投资的企业完全捕获；
  • 竞争性市场中的企业按私人收益率做决策，而社会最优要求按社会收益率做决策。

模型的核心结论：竞争性均衡路径上的投资率会低于社会最优路径；累积投资的单位成本在竞争性均衡下会高于社会最优。

第三步：政策含义

Arrow 指出了三种可能的修正路径：政府干预（补贴投资）、不完全竞争市场结构下的利润激励，以及不完全信息下的学习过程。他的结论是，在存在知识溢出的情况下，市场失灵是结构性的，而非偶然的。

为什么它重要？

Arrow (1962) 的重要性在于它提供了一个理论起点，此后数十年增长理论沿着它开启的方向不断深化：

与 Solow (1956) 的关系

Solow 的新古典增长模型将技术变迁作为外生变量处理——给定技术进步率，然后研究资本积累的动态路径。Arrow 的贡献在于揭示：如果技术变迁本身是内生的（由投资驱动），那么 Solow 模型中"外生给定"的技术进步率就值得重新审视。Arrow 模型中的"干中学"实际上提供了一种内生化的技术变迁机制。

与 Lucas (1988) 的关系

Lucas 在 Arrow 的基础上将"干中学"扩展为"人力资本积累"机制——不仅来自生产中的经验，也来自正规教育和培训。Lucas 的模型将 Arrow 的洞察系统化为人力资本内生增长的完整理论，是内生增长理论的核心节点。

与 Romer (1986) 的关系

Romer 沿着 Arrow 的知识溢出逻辑，建立了不完全竞争框架下的内生增长模型。他的关键拓展是：知识作为商品具有非竞争性（non-rival）——我的使用不减少你的使用——这使得知识的边际社会收益可以远高于私人收益，导致持续的增长动力。

与 AI 时代的关系

Arrow 关于知识溢出和投资不足的分析，在 AI 时代获得了新的生命力：

  • AI 模型的训练本质上是一种"干中学"过程——模型在处理大量任务的过程中不断优化；
  • AI 能力的溢出效应远超 Arrow 时代的制造业——一个公司训练的模型可以低成本地传播到整个行业；
  • AI 投资的社会收益率可能远高于私人收益率，这为政府对 AI 基础研究的支持提供了理论依据；
  • AI 的路径依赖——数据和网络效应带来的自我强化——与 Arrow 关于累积投资的路径依赖逻辑高度一致。

如何阅读这篇论文？

建议分三个阶段阅读，总耗时约 30–60 分钟：

第一遍：抓核心（15 分钟）

先读第 1 节（Introduction）和第 5 节（Conclusions）。第 1 节清晰说明了 Arrow 要回答的问题和研究动机；第 5 节直接给出了核心结论和政策含义。这两节足以把握论文的主心骨。

第二遍：跟论证（20–30 分钟）

再读第 2 节（The Model）和第 3 节（Economic Implications of Learning by Doing）。重点理解 Arrow 如何将"干中学"形式化为一个数学模型——关键不是公式本身，而是 Arrow 的逻辑假设：他为什么这样设定生产函数？溢出效应是如何引入的？结论如何从假设推导出来？

第三遍：读原文细节（20 分钟）

最后通读全文，关注 Arrow 的行文风格和论证细节。注意他如何将经验观察与理论模型交织在一起，以及他如何在结论部分处理模型未能涵盖的现实复杂性。

💡 阅读提示

  • 这篇论文的数学不难——关键在于理解 Arrow 的概念设定（conceptual setup）；
  • 注意区分"竞争性均衡路径"和"社会最优路径"的比较——这是论文的核心推理；
  • 论文中提到"gross investment"和"replacement investment"的区别对于理解模型至关重要；
  • 第 3 节的数学推导较长，可以先跳过公式，重点理解每段文字说明了什么结论。

对 AI agent 工作流的启发

Arrow (1962) 的逻辑框架与现代 AI agent 的工作流存在深层对应关系，理解这种对应关系有助于设计更好的 agent 系统：

经验积累 → 能力提升

Arrow 的"干中学"：投资带来经验积累，经验提升生产率。
AI agent 的类比：每次任务执行（prompt + response）产生经验数据，这些数据用于微调或 context refinement，使 agent 能力持续提升。

知识溢出 → 跨任务泛化

Arrow 的"知识溢出"：一个企业的经验惠及其他企业。
AI agent 的类比：一个任务中学习的模式、推理结构泛化到其他相关任务——这是 in-context learning 的本质。

投资不足 → RLHF 的必要性

Arrow 的"市场失灵"：私人投资低于社会最优，因为企业无法完全捕获知识溢出的收益。
AI agent 的类比：如果只靠 prompt engineering（私人投入），agent 能力的社会收益无法被充分挖掘——这解释了为什么需要 RLHF（人类反馈强化学习）作为系统性投资来弥补市场失灵。

路径依赖 → 能力长尾效应

Arrow 的"路径依赖"：过去的累积投资决定当前生产率，发达与落后之间的差距会自我强化。
AI agent 的类比：高质量训练数据的累积（pre-training + fine-tuning）决定了 agent 的能力基线，数据丰富的领域（代码、对话）能力提升更快，数据稀缺的领域则持续落后。

核心术语简表

干中学 Learning by Doing

通过生产活动中的经验积累获得知识和技术改进，而非刻意研发。

技术变迁 Technological Change

生产函数中从投入到产出的效率提升。Arrow 的贡献在于将其内生化。

体现型技术变迁 Embodied Technical Change

技术进步内嵌于资本品中——新机器包含比旧机器更先进的技术，与资本积累同步发生。

内生增长 Endogenous Growth

增长理论的一个分支，试图从经济系统内部（而非外生给定）解释持续增长。

知识溢出 Knowledge Spillover

一个经济主体创造的知识部分地被其他主体免费获取，产生正外部性。

社会最优 Social Optimum

考虑所有主体（包括受溢出影响的外部主体）时的最优资源配置，与竞争性均衡不同。

总投资 Gross Investment

包括替换投资（replacement）和净增投资（net addition to capital）。Arrow 特别强调两者对知识积累的不同作用。

资本—劳动比 Capital-Labor Ratio

每个工人配备的资本量，是决定工资和利率的关键相对价格。
