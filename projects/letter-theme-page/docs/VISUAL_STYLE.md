# VISUAL_STYLE.md — 信抵达之前

## 页面视觉气质

整体风格定位：**明亮、现代、有纸质感，避免过度昏黄复古**。

这不是一个"旧信纸"主题的怀旧网站，而是一个以当代视角审视书信母题的**数字档案馆**。视觉语言需要同时传达两种质感：
- **档案的严肃性** — 清晰的网格、克制的配色、充足的留白
- **信件的亲密感** — 纸的纹理、手写的痕迹、柔和的阴影

核心视觉隐喻：**光线穿过信纸**。想象一下：午后阳光从窗户斜射进来，照在一张半透明的信纸上，纸的纤维、折痕、甚至背面的字迹都隐约可见。这就是页面要营造的氛围——明亮、通透、有层次。

### 使用规则
- 背景始终使用 `#F7F5F0`，纯白仅用于卡片内部高亮区域
- 母题色仅用于标签、图标和图谱节点，不用于大面积背景
- 强调色（邮戳蓝/火漆红）遵循 60-30-10 原则：60% 中性色、30% 主色、10% 强调色
- hover 状态使用透明度变化（如 `opacity: 0.8`）而非改变色值

## 字体建议

### 中文字体栈
```css
font-family: "Noto Serif SC", "Source Han Serif SC", "STSong", "SimSun", serif;
```
- 标题使用宋体/明朝体，传达书籍与档案的质感
- 正文也可以使用同一字体族，但字重稍轻

### 西文字体栈
```css
font-family: "Cormorant Garamond", "Georgia", "Times New Roman", serif;
```
- 与中文宋体风格协调的衬线体
- 用于英文标题、年份、quote

### 等宽字体（用于邮戳、日期、档案编号）
```css
font-family: "Courier Prime", "Courier New", monospace;
```

### 字体加载策略
- 使用 Google Fonts 或本地 CDN
- 提供系统字体 fallback，确保中文环境快速渲染
- `@font-display: swap` 避免 FOIT

### 字号层级
| 层级 | 桌面端 | 移动端 | 用途 |
|------|--------|--------|------|
| H1 | 48px | 32px | Hero 主标题 |
| H2 | 32px | 24px | 板块标题 |
| H3 | 24px | 20px | 卡片标题 |
| Body | 16px | 16px | 正文（移动端不小于 16px） |
| Caption | 14px | 13px | 标签、元信息 |
| Small | 12px | 12px | 档案编号、邮戳 |

## 图像策略

### 原则
1. **避免直接使用电影海报** — 版权风险高，且视觉风格不统一
2. **优先抽象表达** — 用视觉元素暗示信的存在，而非展示信的具体内容
3. **统一视觉语言** — 所有图像遵循同一套插画风格

### 图像类型优先级

| 优先级 | 类型 | 说明 |
|--------|------|------|
| 1 | SVG 抽象插画 | 每部作品配一张极简 SVG，用几何图形暗示母题 |
| 2 | 统一图标系统 | 8 个母题各有专属图标，使用 SVG 路径 |
| 3 | 纹理与装饰 | 信纸纤维、邮戳、火漆印、折痕等作为背景纹理 |
| 4 | 公共领域图像 | 仅用于档案类条目（如侨批原件、历史照片），需标注来源 |
| 5 | 色块占位 | 无图像时的降级方案：色块 + 作品首字母 |

### SVG 插画方向

为每个母题设计统一的视觉语言：

**代写之信**：两支交叉的羽毛笔，笔尖滴落的墨水分成两股
**迟到之信**：一个信封上叠加逐渐淡化的时钟轮廓
**亡者之信**：一封未拆封的信，信封上别着一片半透明的花瓣
**移民之信**：信封的轮廓中嵌入简化的海路线图
**战争之信**：一张薄纸上印有被涂黑的文字块，光线从背面透过来
**未寄之信**：空白信封半开，里面透出数字屏幕的冷光
**送信者**：蜿蜒的山路上有一个移动的圆点，两端各有一个信箱
**信件档案**：打开的档案盒中，信件整齐排列成网格

### 纹理素材
- 信纸纹理：使用 subtle 的纸张纤维 SVG pattern，不喧宾夺主
- 邮戳：圆形或椭圆形的日期戳，使用等宽字体
- 火漆印：仅用于装饰性元素，不作为功能按钮
- 折痕：卡片的 subtle 对角线渐变，模拟纸张折叠

## 动画与交互风格

- **缓动函数**：主要使用 `cubic-bezier(0.4, 0, 0.2, 1)`（Material Design 标准缓动）
- **过渡时长**：
  - 微交互（hover、点击）：150-200ms
  - 卡片展开/收起：300ms
  - 页面板块切换：400ms
  - 时间线节点激活：500ms
- **SVG 动画**：仅使用 CSS transition/animation，不引入复杂 JS 动画库
- **减少动效**：提供 `prefers-reduced-motion` 媒体查询支持

## 版权与来源注意事项

### 高风险内容（避免使用）
- 电影海报、剧照、蓝光封面
- 出版社书籍封面扫描
- 商业图库的水印图片
- 社交网络用户上传的截图

### 低风险内容（可以使用）
- 自行绘制的 SVG 插画
- 公共领域（Public Domain）的历史图像
  - 美国国家档案馆（NARA）
  - 国会图书馆（LOC）
  - Wikimedia Commons（需检查具体许可）
- Creative Commons 许可图像（需遵循署名要求）
- 抽象几何图形和纹理（无版权）

### 侨批相关图像使用建议
- 优先使用档案馆发布的数字化图像（如 NARA、潮汕历史文化研究中心）
- 必须标注来源档案馆名称
- 涉及个人姓名和地址的侨批，建议做模糊处理以保护隐私
- 可考虑使用"侨批文字局部特写"而非整张扫描，既规避隐私又增强设计感

### 标注规范
所有非原创图像必须在页面底部或模态框内提供：
- 来源机构名称
- 获取 URL
- 许可类型
- 如为 CC 许可，附作者署名

## 技术实现建议

- 使用 CSS 变量管理颜色，便于主题切换
- SVG 插画内联在 HTML 中，便于 CSS 控制颜色
- 考虑使用 `mix-blend-mode: multiply` 让邮戳纹理自然叠加在背景上
- 纸张纹理使用 base64 内联 SVG 或 CSS gradient，减少 HTTP 请求
