Arrow vs. Solow / Lucas / Romer

从四个维度对比 Arrow (1962) 与后续内生增长理论先驱的思想脉络

表一：生产函数与稳态增长对比

模型生产函数技术变迁类型稳态增长率是否内生
Arrow (1962) $x = r(G) - r(G')$ 体现型（嵌入资本品） $\dot{G}/G = \alpha/(1-n)$ ✅ 完全内生
Solow (1956/57) $Y = F(K, AL)$ 非体现型（外生 $A(t)$） $g^* = g$（外生） ❌ 完全外生
Lucas (1988) $Y = AK^\alpha(hL)^{1-\alpha}$ 人力资本体现型 $g^* = (1-\alpha)\mu^{1-\nu}u^{*1-\nu} - \delta$ ✅ 内生
Romer (1990) $Y = H_Y^{1-\alpha}\int_0^A(A_iL_i)^\alpha di$ 非体现型（研发驱动） $g^* = \delta H_A/A$ ✅ 内生
关键洞察：Arrow (1962) 是第一个用形式化模型证明"增长完全内生"的先驱。 Solow 模型中储蓄率不影响长期增长率（外生 $g$），而 Arrow 模型中 $\dot{G}/G = \alpha/(1-n)$ 完全由内生参数决定。

表二：知识累积机制与外部性对比

模型知识载体累积渠道外部性来源外部性方向
Arrow (1962) 体现于资本品的知识 投资的副产品（干中学） 投资的知识溢出 $r(G) < A(G)$
Solow (1956) 技术指数 $A(t)$ 外生给定（无机制） 无 无
Lucas (1988) 人力资本（依附于人） 学校教育 + 干中学（双轨） 人力资本溢出 私人 < 社会
Romer (1990) 知识（想法，非竞争性） 专门研发部门生产 知识非竞争性 私人 < 社会

表三：市场失灵与政策含义对比

模型市场失灵类型竞争性均衡社会最优政策含义
Arrow (1962) 投资不足（知识溢出） $W < \gamma$ $\gamma$（投资率） 补贴 R&D / 产业政策
Solow (1956) 无市场失灵 $g^* = g$ $g^* = g$ 无（储蓄率不影响增长率）
Lucas (1988) 人力资本投资不足 $g_c < g^*$ $g^*$ 补贴教育 / 干中学激励
Romer (1990) 研发投入不足 $g_c < g^*$ $g^*$ 专利保护 / 补贴研发

Arrow → Lucas → Romer：思想演进图

Arrow (1962)

干中学 = 投资驱动知识积累
外部性 = $r(G) < A(G)$，$W < \gamma$
规模效应 = 有

→
Lucas (1988)

人力资本 = 学校教育 + 干中学（双轨）
外部性 = 人力资本溢出
规模效应 = 无

→
Romer (1990)

知识 = 研发部门专门生产
外部性 = 知识非竞争性
规模效应 = 正向（无限增长可能）

关键演进：Arrow 的"干中学"是第一个内生技术变迁机制，但知识溢出是"附带的"（投资的副产品）。 Lucas 将学习渠道显式化（学校+干中学），Romer 则将知识生产完全独立为专门部门。 三者的共同点：都存在外部性导致的市场失灵，都支持产业政策。
