# 活动重设计风险审核表

**文档性质：** 研究型模拟审核工具 | **不代表 OPPO 官方立场**

---

## 审核方法

每项风险按以下维度评估：
- **风险描述**：该风险的具体含义
- **原活动风险**：原 OPPO 母亲节传播中对应风险的严重程度
- **重设计规避方式**：本方案采取的具体规避措施
- **审核问题**：审阅者应关注的具体问题
- **是否通过**：🟢 通过 / 🟡 待确认 / ❌ 需修改

---

## 风险审核表

### 1. 性别风险

| 维度 | 内容 |
|------|------|
| **风险描述** | 表达方式可能矮化女性、将女性客体化或强化刻板印象 |
| **原活动风险** | 🟠 高 — "老公梗"将母亲"自我"绑定于男性关系框架，客观上矮化了母亲的主体性 |
| **重设计规避方式** | 完全不使用任何涉及男性关系框架的符号；"热爱"定义为"个人兴趣/精神追求"，与任何关系框架无关 |
| **审核问题** | 海报文案中是否出现任何将女性的自我实现与男性关系绑定的表达？ |
| **是否通过** | 🟢 通过 |

---

### 2. 家庭伦理风险

| 维度 | 内容 |
|------|------|
| **风险描述** | 表达方式可能制造家庭成员之间的对立、竞争或负面比较 |
| **原活动风险** | 🟠 高 — "比爸爸更隆重"等文案在母亲节语境下制造了父母之间的隐性竞争 |
| **重设计规避方式** | 五张海报中不出现任何家庭成员的对比；妈妈的个人热爱不与"家庭付出"形成竞争框架 |
| **审核问题** | 任何一张海报中是否出现了"爸爸"、"丈夫"、"孩子"等家庭成员作为对比框架？ |
| **是否通过** | 🟢 通过 |

### 3. 圈层语言风险

| 维度 | 内容 |
|------|------|
| **风险描述** | 使用特定亚文化圈层（如饭圈、游戏圈）才理解的符号体系，进入公共语境后产生误读 |
| **原活动风险** | 🔴 极高 — "老公"（饭圈偶像昵称）、"穿婚纱"等符号在母亲节公共语境下产生严重伦理冲突 |
| **重设计规避方式** | 五张海报均使用日常生活场景（跑步、写作、摄影、聚会、音乐），无任何亚文化圈层特有符号 |
| **审核问题** | 文案中是否出现了任何特定圈层（如饭圈、偶像、游戏）的特有词汇或表达？ |
| **是否通过** | 🟢 通过 |

### 4. 饭圈误读风险

| 维度 | 内容 |
|------|------|
| **风险描述** | 将妈妈的个人热爱与"追星"、"偶像"关联，引发饭圈文化与主流家庭价值的冲突 |
| **原活动风险** | 🔴 极高 — 直接使用"老公梗"，在母亲节语境下产生严重语义冲突 |

| **是否通过** | 🟢 通过 |

### 5. 父亲形象风险

### 6. 母亲刻板印象风险

| 维度 | 内容 |
|------|------|
| **风险描述** | 表达方式可能强化"好母亲"的标准，或将母亲神圣化/牺牲者化 |
| **原活动风险** | 🟡 中 — 原活动虽然挑战了母亲刻板印象，但表达方式（"老公梗"）反而可能强化"母亲应该顾家"的暗示 |
| **重设计规避方式** | 五张海报呈现的母亲形象多元、自然；不以"牺牲"或"无私奉献"为叙事核心；不以"完美的母亲"为标准 |
| **审核问题** | 海报中的母亲形象是否出现了"必须"、"应该"的标准框架？是否有"好母亲应该X"的信息？ |
| **是否通过** | 🟢 通过 |

### 7. 年龄羞辱风险

| 维度 | 内容 |
|------|------|
| **风险描述** | 表达方式对特定年龄段的女性进行贬低或玩笑（如"广场舞大妈"、"老阿姨"等） |
| **原活动风险** | 🟢 低 — 原活动未涉及年龄羞辱 |

| **审核问题** | 文案/画面中是否出现了年龄羞辱或年龄焦虑相关的表达？ |
| **是否通过** | 🟢 通过 |

### 8. 隐私风险

| 维度 | 内容 |
|------|------|
| **风险描述** | 海报或文案中可能涉及个人隐私信息，或引导用户传播他人隐私 |
| **原活动风险** | 🟠 高 — 原活动引发舆情后，涉事个人相关信息被大量传播；虽然没有直接证据显示OPPO参与了传播，但品牌方在舆情中未做反隐私保护 |
| **重设计规避方式** | 五张海报均为抽象人物/通用生活场景，不涉及任何真实个人；本方案不使用真实照片；社交活动引导用户拍摄"自己的妈妈"，而非"别人的妈妈" |
| **审核问题** | 任何素材中是否出现了可识别的个人姓名、手机号、地址、学校等隐私信息？ |
| **是否通过** | 🟢 通过 |

| **审核问题** | 文案中是否出现了以代际差异制造话题的倾向？ |
| **是否通过** | 🟢 通过 |

### 10. 平台舆论风险

| 维度 | 内容 |
|------|------|
| **风险描述** | 传播内容在微博/小红书等平台引发负面舆情或被恶意解读 |
| **原活动风险** | 🔴 极高 — 原活动直接引发大规模舆情，并扩散至对个人的开盒和人肉 |
| **重设计规避方式** | 从源头规避高风险符号；建立传播前风险审核清单；备有舆情应对预案 |
| **审核问题** | 本方案的传播路径设计中是否包含舆情监控和快速响应机制？ |
| **是否通过** | 🟢 通过（需配合传播路径设计中的预案） |

### 11. 广告法 / 合规风险

| 维度 | 内容 |
|------|------|
| **风险描述** | 文案或视觉素材违反《广告法》相关规定（如绝对化用语、虚假宣传、代言人规定等） |
| **原活动风险** | 🟡 中 — 原活动本身未涉及明显违法，但舆情扩大后可能引发监管关注 |
| **重设计规避方式** | 文案不包含绝对化用语（"最好"、"第一"等）；产品功能描述真实；不使用"国家级"、"最高级"等禁用词 |
| **审核问题** | 文案中是否出现了任何广告法禁用词汇或虚假宣传表述？ |
| **是否通过** | 🟢 通过 |
