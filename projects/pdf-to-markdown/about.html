About This Project

PDF Processing — First public release

What This Is

A project to turn PDF documents into online, copyable, searchable resources. Each item provides three formats:

  • HTML — single-file, works offline, images embedded as data URIs
  • Markdown — clean text with structure, suitable for editing or archiving
  • Text — plain extracted text for quick reference

Current Scope

The first public release includes 4 digital PDFs. These are documents with embedded text layers — the most reliable path for accurate extraction.

Scanned documents are not included in this public release. The scanned PDF pipeline (OCR + image-first HTML) has been validated internally, but output quality — particularly for older Chinese-language scans — is still under review and improvement.

Current Status

  • v1 — First public release
  • 4 digital PDFs processed and published
  • Dual pipeline (digital + scanned) built and tested
  • Published via GitHub Pages

Future iterations will expand the collection and refine processing methods.
