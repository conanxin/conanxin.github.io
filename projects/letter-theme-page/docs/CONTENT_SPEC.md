# CONTENT_SPEC.md — 信抵达之前

## 页面核心论点

《给阿嬷的情书》中最动人的不是写信动作本身，而是侨批如何同时承载五种功能：通信、汇款、家庭责任、善意谎言、历史档案。书信在文艺作品中从来不只是信息传递工具——它是时间的容器、情感的化石、人际关系的棱镜。

本项目以电影《给阿嬷的情书》为 Hero 引子，构建一个"文艺作品中的书信母题档案馆"。用户通过母题筛选、作品卡片、信件旅程时间线、关系图谱和阅读路线，探索"信"在不同媒介中的多重面貌。

## 八个母题说明

### 1. 代写之信
**核心张力**：写信人与署名者不是同一人。信的真实性、情感归属与伦理边界成为叙事焦点。
**视觉线索**：两支笔、镜像文字、代笔人背影

### 2. 迟到之信
**核心张力**：时间差成为情感放大器。信抵达时，收信人或寄信人已经改变。
**视觉线索**：泛黄的邮戳、时钟叠影、季节变迁

### 3. 亡者之信
**核心张力**：信成为生者与逝者之间的桥梁。写完信的人已经不在，但字迹犹温。
**视觉线索**：羽毛笔、飘落的花瓣、未拆封的信封

### 4. 移民之信
**核心张力**：地理距离被迫转化为文字距离。家书中的经济汇报与情感表达相互缠绕。
**视觉线索**：海路线图、汇率数字、双语文字

### 5. 战争之信
**核心张力**：生死未卜之间的有限字符。审查、涂黑、无法言说的部分比写下的更重。
**视觉线索**：军邮戳、涂改痕迹、薄纸

### 6. 未寄之信
**核心张力**：信存在，但从未进入邮政系统。抽屉里的信、删除的邮件、未发送的草稿。
**视觉线索**：空白信封、抽屉缝隙、数字光标闪烁

### 7. 送信者
**核心张力**：信的内容被第三方知晓或误读。送信人的选择、延迟、甚至篡改改变故事走向。
**视觉线索**：邮差背影、山路、信箱

### 8. 信件档案
**核心张力**：私人通信如何成为公共记忆。从个人抽屉到博物馆展柜，信的语境被重写。
**视觉线索**：档案盒、编目标签、玻璃展柜反光

## 首批作品条目设计

每个作品条目包含以下字段：
- `id`：唯一标识
- `title`：作品名称（原文 + 中文）
- `year`：发行/出版年份
- `medium`：媒介类型（film / novel / animation / drama / documentary / project / historical）
- `motifs`：关联母题数组（1-3 个）
- `pitch`：一句话定位（30 字以内）
- `letter_desc`：信中关键场景描述（80-120 字）
- `quote`：作品中与信相关的代表性台词或文本片段
- `context`：历史或创作背景（可选，50-80 字）
- `region`：国家/地区
- `image_strategy`：图像处理建议（svg / placeholder / public_domain / text_only）

## 母题示例文案

### 引子：《给阿嬷的情书》
> 一封从泰国寄往潮汕的侨批，承载着汇款、谎报平安的善意谎言，以及一个儿子对母亲无法当面说出口的爱。这不是普通的家书——它是整个离散华人家庭的情感基础设施。

### 代写之信：《紫罗兰永恒花园》
> 自动手记人偶薇尔莉特为他人代写书信，却在别人的情感中逐渐理解"我爱你"三个字的分量。代写之信的核心悖论：你必须先成为他人，才能学会成为自己。

### 迟到之信：《情书》
> 博子在未婚夫三周年忌日寄出一封发往天国的信，却收到同名同姓女孩的回信。两封信在时空中交错，揭开一段从未被告白的暗恋。岩井俊二用雪与信证明：有些爱需要迟到才能被看见。

### 亡者之信：《给桃子的信》
> 父亲去世后，桃子在他留下的信中发现只写了三个字："给桃子"。未完成的信成为父亲留在人间最后的锚点，也是女儿学会放下的起点。

### 移民之信：《布鲁克林》
> 1950 年代，爱尔兰女孩艾莉丝寄回的家书必须在报喜与报忧之间小心平衡。信纸上的墨迹厚度，暗示着她正在两个国家之间被撕裂。

### 战争之信：《硫磺岛来信》
> 克林特·伊斯特伍德从日军视角重述太平洋战争。将军西竹一的信件在战败后被美军发现，信中写的不是军令，而是对妻子和马的思念。

### 送信者：《那山那人那狗》
> 湖南乡邮员退休，儿子第一次独自走那条山路。父亲不放心，悄悄跟在身后。信的内容从未被展开，但父子之间未说出口的理解，在这条送信路上终于抵达。

### 未寄之信：《秒速五厘米》
> 贵树在暴风雪中前往与明里约定的车站，随身携带的信在风雪中被吹散。那封没有寄达也没有被读到的信，成为整个青春期最精确的隐喻。

### 信件档案：Letters of Note
> 一个收集"值得被保存的信件"的网站。从甘地写给希特勒的和平呼吁，到某公司员工辞职信——这些被档案化的通信证明：最私人的书写，往往最具公共价值。
