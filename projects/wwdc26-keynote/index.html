开发者层：AI 基础设施

AI 系统架构

点击节点展开说明

Foundation Models framework

原生 Swift API，直接访问 Apple Intelligence 设备端模型。支持 Apple Foundation Models、Claude、Gemini 或其他符合 Language Model protocol 的模型。

  • 访问 Apple Intelligence on-device model
  • Multimodal prompts（图像+文字）
  • Dynamic Profiles 动态切换
  • Private Cloud Compute（Small Business Program 免费）

设备要求：iPhone Air / iPhone 17 Pro+ / iPad M4 12GB+ / Mac M3 12GB+ / Vision Pro M5

App Intents framework

让第三方 App 接入 Apple Intelligence 和 Siri AI。通过 schemas 让 App 内容可被自然语言发现和调用。

  • Entity schemas 贡献内容到 Spotlight 语义索引
  • Intent schemas 让自然语言操作 App 内容
  • View Annotations API 提供屏幕感知
  • App Intents Testing framework 验证集成

Core AI

OS 内置框架，为 Apple Silicon 设计，完全设备端运行模型，无服务器依赖、无 token 成本、保护用户数据。

  • 自动为运行硬件专门化模型
  • ahead-of-time 编译快速加载
  • 细粒度推理内存控制
  • 零拷贝数据路径、有状态执行
