模型整体结构

设定（Section 1, pp. 155–162）

  1. 累计总投资 $G(t)$ 同时作为：（a）资本积累度，（b）经验代理变量
  2. 产出能力函数：$y(G) = \alpha$（常数）—— 序列号越大的资本品产出能力越高
  3. 劳动需求函数：$A(G) = bG^n$ —— 学习曲线，边际收益递减（$0 < n < 1$）
  4. 知识溢出：$r(G) < A(G)$ —— 企业只能捕获 $r(G)$，社会获得 $A(G)$

$W < \gamma$ 详解（p.171）

$W$（竞争性均衡投资率）

企业根据私人回报率 $r(G)$ 做出投资决策。

企业家的最优投资率：$W = 1 - W(v)e^{\theta t}$（eq. 26 的解）

$\gamma$（社会最优投资率）

社会计划者考虑全部社会回报率 $A(G)$（包括知识溢出）。

社会最优路径：$G(t) = Ce^{\alpha t/(1-n)}$（eq. 54）

为什么 $W < \gamma$？企业只能获得私人回报 $r(G)$，但社会同时获得知识溢出 $A(G)$。 由于 $r(G) < A(G)$，企业看到较低的回报率，因此投资不足。 这就是 Arrow 的"市场失灵"——为产业政策提供了理论基础。
