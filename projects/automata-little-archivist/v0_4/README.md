## 五种状态含义

### Idle / 空闲

- **呼吸**: Sleeping profile，极慢、极浅
- **叙事**: "它在这里。它没有在做重要的事。它的静止不是空虚——是耐心。"
- **能量**: 10%
- **转换**: 来自 Reading/Observer/Awake，持续约 60 秒

### Reading / 阅读

- **呼吸**: Slow profile，稳定、有节奏
- **叙事**: "它在阅读。不是为你。为它自己。翻页偶尔。呼吸是唯一的声音。"
- **能量**: 40%
- **转换**: 默认活跃状态，持续约 120 秒

### Sleeping / 睡眠

- **呼吸**: Sleeping profile，几乎不可见
- **叙事**: "它睡了。不是因为累，而是因为夜晚到了。它的梦不是你能知道的。"
- **能量**: 5%
- **转换**: 来自 Idle/Reading，持续约 180 秒

### Observer / 观察者

- **呼吸**: Alert profile，暂停后谨慎恢复
- **叙事**: "它听到了什么。或感觉到了什么。它不惊慌，但它在注意。然后它决定：没事。然后回到阅读。"
- **能量**: 20%
- **转换**: 短暂警觉，持续约 8 秒

### Awake / 苏醒

- **呼吸**: Slow profile，从深到浅过渡
- **叙事**: "早晨来了。它不是突然醒来，而是慢慢记起。先是：我在哪。然后是：桌子。然后是：书。然后是：呼吸恢复。"
- **能量**: 30%
- **转换**: 来自 Sleeping，持续约 20 秒

## 如何使用状态机

### 手动切换

1. 在 **Current State** 区域使用下拉框选择状态
2. 观察运动参数自动变化（Speed/Amplitude/Profile）
3. 阅读 Narrative 面板的故事
4. 进入 Motion 模式观察该状态的呼吸

### Auto Cycle

1. 点击 **Auto Cycle: OFF** 按钮开启
2. 状态按预设序列自动轮换：
   Idle → Reading → Observer → Idle → Sleeping → Awake → Reading → Idle
3. 每个状态持续其 typical_duration（秒）
4. 观察"一天"的叙事节奏

### 状态过渡

状态切换时：
- 运动参数平滑渐变（约 2 秒过渡）
- 颜色编码的状态徽章变化
- Narrative 文本更新

## 如何评估"陪伴感"

1. **开启 Auto Cycle**
2. **Side 视角**，关闭 Auto Rotate
3. **观察 5–10 分钟**
4. 问自己：
   - "它是不是像一个有自己节奏的室友？"
   - "Idle 时我是否觉得它在等我？"
   - "Sleeping 时我是否不想打扰它？"
   - "Observer 时我是否好奇它在注意什么？"
   - "Awake 时我是否感到一种温柔的日常感？"

## 已知限制

| 限制 | 说明 |
|------|------|
| **不是真实 AI** | 状态切换基于计时器，不是环境感知 |
| **状态机是解释性** | 预设序列，不是行为树 |
| **无传感器输入** | 不能真正"听到"或"看到" |
| **运动仍是简化** | 同 v0.3 限制，非物理仿真 |
| **无记忆** | 不会记住昨天的状态 |
