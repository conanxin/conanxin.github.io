## 4. 版权与来源合规说明

本阶段遵循以下来源优先级原则：
1. **首选**：作者官方博客/官方网站（本人发布）
2. **次选**：可信学术存档（大学档案、Internet Archive）
3. **不采用**：低质量 SEO 站、随机搬运站、无来源说明的复制页
4. **不可嵌入**：纯文章类条目无需 embed_url；如原内容是视频/音频，则需找到合法可嵌入版本后才填 embed_url
