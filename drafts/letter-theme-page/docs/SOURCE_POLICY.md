# 来源策略文档

## 项目定位

本页为个人阅读与研究型互动索引。所有作品信息基于公开资料整理，不声称学术权威性，但追求可追溯与可核查。

## 来源优先级

| 优先级 | 来源类型 | 示例 |
|--------|----------|------|
| 1 | 官方网站 / 发行方 / 出版社 | 电影官网、出版社书目页、剧院公告 |
| 2 | 权威文化机构 / 博物馆 / 档案馆 | UNESCO、国家档案馆、图书馆数字馆藏 |
| 3 | 权威百科全书 / 学术数据库 | Britannica、JSTOR、Project MUSE |
| 4 | 影展 / 电影节官方资料 | 戛纳、柏林、威尼斯、金马影展 |
| 5 | 综合影视数据库 | IMDb、Letterboxd（仅作辅助线索） |
| 6 | 中文综合数据库 | 豆瓣电影（标注为数据库来源，非权威审定） |
| 7 | 项目自述 / 创作者公开访谈 | 博客、采访（标注为自述来源） |

## 来源状态定义

| 状态 | 含义 | 使用场景 |
|------|------|----------|
| `verified` | 已通过官方或权威来源核实关键信息 | 有官网、出版社页或权威百科 backing |
| `partial` | 部分信息已核实，部分基于公开常识 | 年份/导演已确认，但具体细节未逐一核对 |
| `placeholder` | 暂未开始核实 | 新添加条目或待处理项 |
| `needs_review` | 资料可能有争议或需要人工复核 | 年份存疑、作者信息不完整、或来源有冲突 |
| `not_found` | 暂未找到可靠来源 | 条目过冷门、语言障碍、或原始资料已丢失 |

## 禁止事项

1. **不要编造**上映年份、导演、编剧、剧情细节
2. **不要把二手博客**当作唯一来源
3. **不要直接使用电影海报图片**——本页所有视觉均为原创 SVG 或占位
4. **不要复制长篇版权文本**——简介控制在 160 字以内，引用台词控制在合理 fair use 范围
5. **不要标注未经核实的来源为 verified**——如果不确定，宁可降级为 partial 或 needs_review

## 核实流程

```
新增条目 → 标注 placeholder → 尝试查找官方/权威来源
    → 找到可靠来源 → 标记 verified，记录 URL
    → 找到部分信息 → 标记 partial，记录已核实项
    → 找不到来源 → 标记 not_found 或 needs_review
    → 信息有冲突 → 标记 needs_review，记录冲突点
```

## 图像与版权策略

- 当前所有图像均为**原创 SVG**（母题图标、路线装饰图）
- 不使用任何电影海报、剧照、书籍封面扫描
- 如需引入历史档案图像（如侨批原件），优先使用：
  - 公共领域（Public Domain）资源
  - Creative Commons 许可资源（遵守署名要求）
  - 档案馆官方数字化图像（标注来源机构）
- 涉及个人隐私的侨批需做模糊处理
