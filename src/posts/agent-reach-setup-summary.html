🔧 可选配置（按需开启）

配置 Twitter/X（需要 Cookie）

步骤:
  1. 浏览器登录 https://x.com
  2. 安装 Cookie-Editor
  3. 获取 auth_token 和 ct0 值
  4. 设置环境变量:
   export AUTH_TOKEN="你的token"


   export CT0="你的ct0"
  1. 验证: bird check

配置 Reddit（需要代理）

说明: 服务器 IP 被 Reddit 封锁，需要住宅代理 步骤:
  1. 购买住宅代理 (Bright Data, IPRoyal 等)
  2. 配置代理:
   agent-reach configure proxy "http://user:pass@host:port"

配置小红书（需要 Docker）

步骤: 
<h1>1. 安装 Docker</h1>


sudo apt install docker.io



<h1>2. 运行小红书 MCP</h1> docker run -d --name xiaohongshu-mcp \   -p 18060:18060 \   xpzouying/xiaohongshu-mcp



<h1>3. 配置 mcporter</h1> mcporter config add xiaohongshu http://localhost:18060/mcp
