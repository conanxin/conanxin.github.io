# 「侨批与南洋」专题小章设计

## 设计意图

《给阿嬷的情书》表面上是一部关于「情书」的纪录片，但它的核心不是爱情，而是**侨批**——一种在 19 世纪至 20 世纪中叶流行于东南亚华人与中国故乡之间的特殊通信方式。

理解侨批，才能理解为什么「信」在这个语境中不只是文字，而是：
- **生活费**（汇款单上的数字）
- **平安信号**（「一切安好」的格式化谎言）
- **家庭档案**（代代相传的纸片）
- **信用凭证**（批局与水客之间的信任网络）
- **历史证词**（2013 年入选 UNESCO 世界记忆名录）

## 侨批是什么

侨批（又称「银信」）是海外华人寄回中国的**家书与汇款凭证的合称**。它同时是：

| 功能 | 说明 |
|------|------|
| 通信 | 报平安、述近况、托家事 |
| 汇款 | 通过批局或水客将外币兑换为银元/法币寄回 |
| 信用 | 批局作为中间人，承担汇率风险与运输风险 |
| 档案 | 家庭保存的侨批成为代代相传的「家庭史记」 |
| 情感 | 「报喜不报忧」的表演性书写，隐藏真实处境 |

## 与现有母题的关系

侨批不是一个独立的母题，而是**移民之信（migration）+ 信件档案（archive）的交集**。但它值得一个专门的页面小章，因为：

1. 它是《给阿嬷的情书》的理解起点
2. 它涉及具体的历史机制（批局、水客、邮路）
3. 它有真实的物质形态（信纸、汇款单、邮戳）
4. 它已经获得了世界性的文化认可（UNESCO）

### 交互点 1：侨批结构拆解

一个可点击的「解剖图」，展示一封侨批的组成部分：

- **信纸**：手写家书，通常是格式化问候（天气、健康、平安）
- **汇款单**：批局开具的凭证，显示金额、汇率、手续费
- **地址**：故乡的详细地址，通常精确到村、族、房号
- **邮路**：泰国/马来 → 汕头/厦门/泉州，经水客或批局
- **报平安**：「一切安好」的标准化谎言，隐藏真实困境

交互方式：点击每个部分，展开 50-100 字的说明。

### 交互点 2：南洋路线

一个简单的 SVG 路线图，展示侨批的物理旅程：

```
故乡（潮汕） ←——批局 ←——水客/船运 ←——海外华人（暹罗/马来/印尼）
     ↓                                              ↑
家庭收信 ←——————————————————————————————————汇款凭证
```

关键节点可点击，显示批局运作、水客角色、汇率机制等说明。

### 交互点 3：从家书到档案

时间线展示侨批的「身份转变」：

- **1850s-1950s**：私人家庭通信，家庭内部保存
- **1960s-1980s**：文革期间大量侨批被销毁或隐藏
- **1990s-2000s**：学者开始收集与研究，进入学术视野
- **2013**：入选 UNESCO 世界记忆名录，成为「人类共同遗产」
- **2020s**：《给阿嬷的情书》等纪录片将侨批带入大众文化

## 推荐文案草稿

### 小章标题
> 侨批：不只是情书

### 核心段落（80-120 字）
> 阿嬷收到的不是普通情书，是侨批。薄薄一张纸，同时是家书、是汇款单、是信用凭证、是家庭档案。信里说「一切平安」，其实儿子在橡胶园里累到吐血；说「不用挂念」，其实三年没见面。侨批的谎言不是欺骗，是离散家庭唯一能负担的温柔。

### 三个交互点的提示文案

**结构拆解**
> 拆开一封侨批，你会看到什么？

**南洋路线**
> 从暹罗到潮汕，一封信要走多久？

**家书变档案**
> 私人信件如何成为世界记忆？

## 风险评估

- **内容风险**：侨批涉及敏感历史（殖民、移民、文革），需避免政治化表述，保持文化史视角
- **版权风险**：侨批原件图片可能涉及博物馆版权，建议使用 SVG 示意图或公开 domain 图片
- **准确性风险**：批局运作细节需要学术来源支撑，不要依赖 Wikipedia 单一来源
