Agent OS 是系统的处理层。它以 Hermes Agent 为核心，通过多平台网关、技能系统、记忆系统和定时任务，把 AI 从「对话框」变成「后台进程」。

$ls ./components
  • Multi-platform Gateway — Telegram / Discord / Slack / WhatsApp / Signal / CLI 统一入口
  • Skill System — 100+ 可插拔技能，覆盖研究、编码、内容创作、MLOps、DevOps
  • Memory & FTS5 Search — 跨会话对话搜索 + LLM 摘要，实现长期记忆回溯
  • Cron Scheduler — 自然语言定时任务，支持日报/备份/审计自动化
  • Subagent Delegation — 并行子代理执行，复杂任务拆解为独立工作流
  • Model Switching — 运行时切换 provider（OpenRouter / Kimi / MiniMax / OpenAI 等）
  • Skin Engine — 数据驱动的 CLI 主题系统，纯 YAML 配置
