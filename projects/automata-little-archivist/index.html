The Little Archivist

小档案员 — Narrative Automata Web Preview

Digital Prototype v0.6 Recommended Three.js / STL / Static HTML

小档案员是一个将 automata 机械自动装置、AI agent 想象、叙事状态机与 3D 打印原型结合的数字机械生命体实验。 它是一个前倾阅读的无面人形，由呼吸凸轮机构驱动，在 5 种叙事状态间切换：Idle、Reading、Sleeping、Observer、Awake。

The Little Archivist is a browser-based narrative automata prototype exploring mechanical presence, breath, and state. A faceless figure leaning forward to read, driven by a breathing cam mechanism, transitioning between 5 narrative states.

v0.6 Shared Assets Demo 是推荐公开展示版本。如果你只想快速了解这个项目，直接打开 v0.6 即可。

Scope & Limitations

  • This is a digital prototype, not a physical device.
  • Motion is interpretive and narrative-driven, not physical simulation.
  • STL files are engineering previews for visualization, not final production assets.
  • No backend required — all rendering happens in-browser via Three.js.
