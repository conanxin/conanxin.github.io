> *思想不是线性的，它们是多维的、相互连接的、不断演化的。*

---

## 这是什么？

这是一个**数字花园**——一种非线性的知识管理系统。

不同于传统博客（按时间线组织、文章一旦发布就固定），数字花园中的笔记是：

- **双向连接**：笔记之间通过 `[[链接]]` 相互引用
- **持续演化**：笔记会随着时间生长、更新、改变
- **主题导向**：按概念而非时间组织
- **过程可见**：展示思考过程，而非仅展示结论

## 如何使用

1. **从 [[index]] 开始**：查看主题导航
2. **跟随链接**：点击 `[[双括号链接]]` 探索相关概念
3. **注意标签**：每篇文章底部的 `#标签` 帮助发现相关内容
4. **查看连接**：`connections/` 目录包含主题的综合分析
